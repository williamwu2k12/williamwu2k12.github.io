Secure Browsing



This is a Google Chrome extension that improves the user's browsing experience. It encrypts sensitive browsing history using CryptoJS.AES. Analytics on link patterns are in development.

TFTI, currently a work in progress, is an iOS application that aims to simplify creating events and distributing invites, making it easier for users to accept, reject, and generally participate (rather than ignore) event invitations.
